# Skill Index

Thư mục này chứa kinh nghiệm tích lũy của Agent trong quá trình làm việc với dự án Chainslake.

> **Hướng dẫn cho Agent**: Đọc index này ở đầu mỗi phiên làm việc. Nếu nhiệm vụ người dùng yêu cầu khớp với một skill đã có, đọc file skill đó và thực thi theo hướng dẫn mà không cần hỏi lại quy trình. Sau mỗi nhiệm vụ thành công, viết/cập nhật skill tương ứng.

---

## add-new-chain-pipeline.md
- **Áp dụng khi**: Thêm một EVM blockchain mới vào hệ thống (ví dụ: BNB, Polygon, Arbitrum, Base, v.v.)
- **Lần cuối cập nhật**: 2026-07-10

## setup-metabase.md
- **Áp dụng khi**: Thiết lập Metabase on-premise từ đầu — tạo admin, kết nối database, tạo API key, setup Metabase CLI
- **Lần cuối cập nhật**: 2026-07-12

## metabase-cli.md
- **Áp dụng khi**: Sử dụng Metabase CLI (`mb`) để quản lý databases, cards, dashboards, collections, transforms, upload CSV
- **Lần cuối cập nhật**: 2026-07-12

## run-dag-and-verify.md
- **Áp dụng khi**: Trigger một Airflow DAG, theo dõi tiến trình, và kiểm tra dữ liệu sau khi chạy xong
- **Lần cuối cập nhật**: 2026-07-11

## upload-csv-to-dwh.md
- **Áp dụng khi**: Upload file CSV vào data warehouse qua HDFS, tạo external table
- **Lần cuối cập nhật**: 2026-07-11

## configure-job-parameters.md
- **Áp dụng khi**: Cấu hình tham số pipeline (`number_block_per_partition`, `max_number_partition`, `max_time_run`, `start_date`, `catchup`, `run_mode`) hoặc backfill job mới
- **Lần cuối cập nhật**: 2026-07-16

## install-chainslake-onprem.md
- **Áp dụng khi**: Cài đặt hệ thống Chainslake Data Warehouse trên máy local hoặc server riêng từ đầu
- **Lần cuối cập nhật**: 2026-07-16

## build-catalog.md
- **Áp dụng khi**: Tạo/cập nhật tài liệu catalog markdown cho toàn bộ bảng trong data warehouse
- **Lần cuối cập nhật**: 2026-07-18
